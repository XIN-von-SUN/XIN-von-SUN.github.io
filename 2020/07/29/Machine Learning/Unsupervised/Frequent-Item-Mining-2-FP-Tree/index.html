Frequent Item Mining 2 - FP Tree

Posted on 2020-07-29 | In Machine Learning , Unsupervised Learning , Frequent Item Mining

As an algorithm for mining frequent itemsets, the Apriori algorithm requires multiple scans of data, and I/O efficiency is a big bottleneck. In order to solve this problem, the FP Tree algorithm (also known as the FP Growth algorithm) uses some tricks, and it only needs to scan the data set twice no matter how much data, thereby improving the efficiency of the algorithm.
