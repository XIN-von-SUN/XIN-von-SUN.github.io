How to develop Python Package as PyPI

Posted on 2020-07-28 | In My Apps development , Python Package PyPI

Why developing own Package

There are a bunch of nifty tools that help you build, install and distribute packages.

Using a well structured, standard layout for our own package makes it easy to use those tools.

Even if you never want to give anyone else your code, a well structured package simplifies development.

What is a Package?

A collection of modules:
… and the documentation
… and the tests
… and any top-level scripts
… and any data files required
… and a way to build and install it…

5. Package and Publish our own Python Package on PyPI

  • run ‘python setup.py check’, continue if no error reported.

  • run ‘python3 setup.py sdist bdist_wheel’ to package all files.

  • run ‘twine upload dist/*’ to publish our own developing package into Python Package PyPI. Before this step, we should register an account in PyPI.

  • Then we can use ‘pip install OurPyPI==.0.1’ to install our own package.
