Loss Function

Posted on 2020-08-04 | In Machine Learning , Machine Learning Foundations

Loss for regression

MSE Loss (L2)
  • Simple and easy to calculate;
  • Euclidean distance is a good measure of similarity;
  • The feature properties remain unchanged after transformation in different representation domains.

MAE Loss (L1)


MAE VS MSE

  • Calculation of MSE is simple, but MAE has better robustness to abnormal points: when there are abnormal points in the data, the model that uses RMSE to calculate the loss will sacrifice the errors of other samples, and move towards reducing the error of abnormal points. However, this will reduce the overall performance of the model.

  • Intuitively, it can be understood like this: if we minimize the MSE to give only one predicted value for all sample points, then this value must be the average of all target values. But if it aims to minimize the MAE, then this value will be the median of the target value of all sample points. As we all know, for outliers, the median is more robust than the mean, so MAE is also more stable for outliers than MSE.

  • The update gradient of MAE in NN is always the same, but MSE is different: the gradient of MSE loss increases as the loss increases, and when the loss tends to 0, it decreases.

  • Loss selection suggestion
    MSE: If the abnormal point represents an abnormal situation that is important in scenaria and needs to be detected.
    MAE: If only outliers are treated as damaged data.

Huber Loss

  • Huber Loss is less sensitive to data outliers than the squared error loss. It can also be guided at 0. Basically it is an absolute error. When the error is very small, the error is quadratic.

  • When the error needs to become a quadratic depends on a hyperparameter $\delta$, which can be fine-tuned. When $\delta \rightarrow 0$, Huber Loss is close to MAE, when $\delta \rightarrow \infty$ (a very large number), Huber Loss is close to MSE.



Loss for classification

0-1 Loss

  • The 0-1 loss function directly corresponds to the number of classification errors, but it is a non-convex function.

Log Loss

  • The log loss function can represent the probability distribution quite well. In many scenarios, especially multi-classification, it is very suitable if you need to know the confidence that the result belongs to each category.

  • The robustness is not strong, and it is more sensitive to noise than Hinge loss.

  • Used in logistic regression.

Hingo Loss (SVM)

  • Hinge loss function is 0 if it is classified correctly, otherwise the loss is $(1 - Yf(x)$. SVM uses this loss function.

  • The $f(x)$ is the predicted value, between -1 and 1, and the $Y$ is the target value (-1 or 1).

  • The robustness is relatively high, and it is not sensitive to abnormal points and noise, but it does not have a good probability explanation.

Perceptron Loss
  • It is a variant of Hinge loss function. The generalization ability of the model is not as strong as hinge loss.

Cross-Entropy Loss

Binary classification:

Multi-class classification:

  • It is also a log-likelihood function essencially, which can be used in binary classification and multi-classification tasks.

  • The cross-entropy loss is often used instead of the mean square error loss function when using sigmoid as the activation function, because it can perfectly solve the problem of slow update of the mse loss weight.

  • When the error is large, the weight update is fast; When the error is small, the weight update is low.

Exponential Loss

  • Quite sensitive to outliers and noise.
  • Often used in AdaBoost algorithm.
